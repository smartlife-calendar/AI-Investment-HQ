# 全方位多模型股市分析系統

**AI-Investment-HQ** 是一個整合多種投資分析框架的股市分析系統，支援美股與台股。

## 🌐 線上試用

👉 **https://smartlife-calendar.github.io/AI-Investment-HQ/**

## 📊 分析框架（7種）

| 框架 | 類型 |
|---|---|
| 財務結構分析 | 現金流、股本稀釋、法說會解讀 |
| 供應鏈結構分析 | 產能瓶頸、垂直整合、時間差 |
| Benjamin Graham | 價值防禦、Graham Number |
| Peter Lynch GARP | PEG ratio、合理成長估值 |
| Cathie Wood 破壞性創新 | 5年CAGR、TAM倍數 |
| Piotroski F-Score | 財務健康量化（0-9分） |
| 技術面分析 | RSI、布林通道、MACD、均線 |

## 📈 數據來源

- **美股**：SEC EDGAR XBRL + Yahoo Finance Chart v8
- **台股**：FinMind + TWSE + Yahoo Finance
- **市場情緒**：VIX、貪婪/恐懼指數、板塊ETF
- **技術指標**：Yahoo Finance 歷史價格自行計算
